# freeCodeCamp Daily Coding Challenge - March 9, 2026

## Array Sum

Given an array of numbers, return the sum of all the numbers.

### Tests:

1. `sumArray([1, 2, 3, 4, 5])` should return `15`.
2. `sumArray([42])` should return `42`.
3. `sumArray([5, -2, 7, -3])` should return `7`.
4. `sumArray([203, 145, -129, 6293, 523, -919, 845, 2434])` should return `9395`.
5. `sumArray([0, 0])` should return `0`.
