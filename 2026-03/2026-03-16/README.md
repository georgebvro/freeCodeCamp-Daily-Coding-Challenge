# freeCodeCamp Daily Coding Challenge - March 16, 2026

## Evenly Divisible

Given two integers, determine if you can evenly divide the first one by the second one.

### Tests:

1. `isEvenlyDivisible(4, 2)` should return `true`.
2. `isEvenlyDivisible(7, 3)` should return `false`.
3. `isEvenlyDivisible(5, 10)` should return `false`.
4. `isEvenlyDivisible(48, 6)` should return `true`.
5. `isEvenlyDivisible(3186, 9)` should return `true`.
6. `isEvenlyDivisible(4192, 11)` should return `false`.
