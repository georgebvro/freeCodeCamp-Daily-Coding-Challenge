# freeCodeCamp Daily Coding Challenge - January 15, 2026

## Array Swap

Given an array with two values, return an array with the values swapped.

For example, given `["A", "B"]` return `["B", "A"]`.

### Tests

1. `arraySwap(["A", "B"])` should return `["B", "A"]`.
2. `arraySwap([25, 20])` should return `[20, 25]`.
3. `arraySwap([true, false])` should return `[false, true]`.
4. `arraySwap(["1", 1])` should return `[1, "1"]`.
